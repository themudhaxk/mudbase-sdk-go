# MUDBASE GO SDK

Auto-generated SDK for MUDBASE API.

## Installation

```bash
go get github.com/themudhaxk/mudbase-sdk-go
```

## Usage

```go
import "github.com/themudhaxk/mudbase-sdk-go"

client := mudbase.NewClient("YOUR_API_KEY")
users, err := client.Users.List()
```
